# ALX Frontend ReactJS Journey

This repository contains a collection of React applications and components developed during the **ALX Frontend Development** course. Each project focuses on core React concepts, API integration, and modern UI styling.

## Projects Included

| Project               | Description                                                       | Tech Highlights                                |
| --------------------- | ----------------------------------------------------------------- | ---------------------------------------------- |
| **Quiz App**          | An interactive quiz with score tracking and logic.                | State management, dynamic rendering.           |
| **Weather Dashboard** | Real-time weather data visualization using APIs.                  | Fetch API, Async/Await, Environment variables. |
| **Portfolio**         | A personal showcase (contained in the `kgn-portfolio` directory). | Tailwind CSS 4, React Router, EmailJS.         |
| _etc._                | _..._                                                             |                                                |

---

## Global Tech Stack

Across these projects, I utilized the following technologies:

- **Core:** React 19, TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS, CSS Modules
- **Routing:** React Router 7
- **State & Data:** React Hooks (useState, useEffect), Axios/Fetch

## How to Run Locally

Follow these steps to run any of the projects in this repository:

1. **Clone the repository:**

```bash
git clone https://github.com/kaberege/kgn-fe-reactjs.git

```

2. **Enter the repository directory:**

```bash
cd kgn-fe-reactjs

```

3. **Navigate into a specific project folder:**

```bash
# Example:
cd kgn-portfolio

```

4. **Install dependencies and start:**

```bash
npm install
npm run dev

```
